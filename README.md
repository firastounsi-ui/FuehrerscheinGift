# Führerschein Gift

A personal learning website for the German driving theory exam.

## Features

- Arabic explanations for German driving theory
- German traffic sign lessons
- Daily quiz with 10 questions
- Question catalogue stored in JSON
- Course content stored in JSON
- Local quiz completion status with localStorage
- Mobile-friendly layout
- Spring Boot backend with static frontend

## Tech Stack

- Java 17
- Spring Boot
- HTML
- CSS
- JavaScript
- JSON
